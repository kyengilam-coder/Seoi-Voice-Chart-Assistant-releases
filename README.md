# Seoi Voice Chart Assistant — 배포 저장소

이 저장소는 **설치 파일과 자동 업데이트 메타데이터만** 공개하는 배포 전용 저장소입니다.
소스 코드와 개발 이력은 별도의 비공개 저장소에서 관리됩니다.

## 설치 / 업데이트
- 최신 설치 파일은 [Releases](../../releases/latest) 에서 받을 수 있습니다.
- 설치 후에는 앱이 자동으로 최신 버전을 확인·업데이트합니다.

## 사용 안내
- 이 앱은 사용자 본인의 OpenAI API 키가 있어야 동작합니다(키는 각 PC에 암호화 저장되며 배포물에 포함되지 않습니다).
